== Set a label

.recon1.js

----
include::20_reconcile/recon1.js[]
----

[source,shell]
----
http PUT localhost:8080/api/v1alpha1/things/default/things/foo/reconciliations 'changed[overTemp][javaScript]=@20_reconcile/recon1.js'
----

Set over temp value:

[source,shell]
----
http PUT localhost:8080/api/v1alpha1/things/default/things/foo/reportedStates temperature:=62
----

Reset with a value below `60`.

== Add as a reference

[source,shell]
----
http POST localhost:8080/api/v1alpha1/things metadata:='{"name": "overTempGroup", "application": "default"}'
----
